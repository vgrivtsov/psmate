Справочная информация

Справочная информация по терминам профессиональных стандартов и сервисам ПрофНавигатора.

Профессиональный стандарт
Профессиональный стандарт – характеристика квалификации, необходимой работнику для осуществления определенного вида профессиональной деятельности, в том числе выполнения определенной трудовой функции (ст. 195.1 Трудового кодекса Российской Федерации (далее – Кодекс)).
Квалификация
Квалификация – уровень знаний, умений, профессиональных навыков и опыта работы работника (ст. 195.1 Кодекса).
Обобщенная трудовая функция
Обобщенная трудовая функция – совокупность связанных между собой трудовых функций, сложившаяся в результате разделения труда в конкретном производственном или бизнес-процессе (приказ Минтруда России № 170н). Каждая обобщенная трудовая функция описывает профессиональную деятельность, которая может быть выполнена одним работником (в большинстве организаций, где реализуется данный вид профессиональной деятельности). При формировании должностной инструкции работника в конкретной организации может быть использован не весь набор трудовых функций в обобщенной трудовой функции или же набор трудовых функций может быть расширен исходя из специфики выполняемой работы. Допускается использовать несколько обобщенных трудовых функций, при этом требования к квалификации работника определяются по более высокому уровню квалификации.
Трудовая функция
Трудовая функция – система трудовых действий в рамках обобщенной трудовой функции (приказ Минтруда России № 170н); В Кодексе (ст. 57) трудовая функция определяется как «работа по должности в соответствии со штатным расписанием, профессии, специальности с указанием квалификации», а также как «конкретный вид поручаемой работнику работы», таким образом, первая часть определения соответствует обобщенной трудовой функции, а вторая – трудовой функции в профессиональном стандарте. Характеристики конкретных трудовых функций содержат описание трудовых действий, необходимых умений и знаний и др.
Поиск профстандарта в ПрофНавигаторе
  • Выберите вкладку Cервисы пункт Найти профстандарт
  • Введите в строке автопоиска интересующее слово из названия профстандарта.
  • Нажмите кнопку показать профстандарт
    • Поиск должности в ПрофНавигаторе
  • Из поиска Квалификационный справочник должностей или из пункта Cервисы навигационной панели, пункт Справочник должностей введите интересующую должность.
  • В списке результатов выберите подходящую пару Должность - Обобщенная трудовая функция Обратите внимание, что название должностей может совпадать в разных профстандартах. В колонке Профстандарт указан номер профстандарта.
  • Наведите курсор на него для получения названия в всплывающей подсказке.
  • Выберете из списка инетресующую должность для получения основной информации.
    • Должностная инструкция в ПрофНавигаторе
  • Найдите должность с помощью Справочника дложностей
  • Выберите соотвествующие трудовые функции(по умолчанию выбраны все)
  • Нажмите кнопку Построить должностные инструкции
  • Исправьте в шапке названия организации, подразделения и пр. - в тексте будут отображаться на введенные(в именительном падеже)
  • Для дополнительных возможностей см. раздел Для зарегестрированных пользователей
    • Для зарегестрированных пользователей

    Построение должностных инструкций для зарегестрированных пользвателей

  • Зайдите в Личный кабинет
  • Зарегестрируйте Организацию
  • Зайдите в созданную Организацию
  • Создайте Подразделение
  • Добавьте должностную инструкцию для соотвествующего Подразделения кнопкой Добавить должностную инструкцию
  • Найдите должность с помощью Справочника дложностей
  • Выберите соотвествующие трудовые функции(по умолчанию выбраны все)
  • Нажмите кнопку Построить должностные инструкции
  • Должностная инструкци сгенерируется в соответствии с настройками организации и подразделения, в правильных падежах по тексту документа.
  • Сохраните должностную инструкцию в подразделении для будущей работы с ней
  • Скачайте должностную инструкцию в формате MS Word (.docx)
    • Создание резюме

  • Выберете в навигационной панели Конструктор резюме
  • Добавьте место работы
  • Нажмите кнопку Добавить квалификацию
  • Выберите отрасль
  • В строке автопоиска введите должность или номер профстандарта или название профстандарта
  • Нажмите Выбрать трудовые функции
  • Будут предложены все трудовые функции профстандарта. Цветом выделены трудовые функции соответсвуюущей обобщенной трудовой функции.
  • Выберите трудовые функции и соотвествующие компетенции
  • Добавьте квалификацию или место работы по желанию.
